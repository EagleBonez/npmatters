Database of Local Housing Need

by Ed Dade

Local Housing Need data for local authority areas in England.

Local Housing Need is the government's standard method for determining the number of new homes required to be built in each local planning authority area. The table below presents the annual Local Housing Need figure for each local authority area for the current year and previous years.

LHN has been calculated using the ONS's 2014-based household projections. The local authorities listed represent those described in the household projections data. Note that some local authorities and their geographies have changed since publication.

For some local authorities, the annual LHN figure may be capped depending on the extent of increase in the housing requirement. No cap has been applied in the data presented.

  • Click a local authority name to view more info about how the LHN has been calculated for that area.
  • Click a 'county' to compare all local authorities in the county.
  • Sort the table by clicking headings.
